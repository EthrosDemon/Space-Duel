3. Einstellungen

3.1 Tasten

Die Tasten können unter dem Menüpunkt "Einstellungen/Tasten..." eingestellt werden.

3.2 Konfiguration

Die Parameter des Spiels sind unter dem Menüpunkt "Einstellungen/Spiel..." einstellbar.

Alle Parameter sind in verschiedenen Konfigurationen zusammengefaßt. Mit der Konfiguration "Benutzerdefiniert" können alle Parameter selbst eingestellt werden. Interessante Konfigurationen bitte an den Autor des Spiels schicken. Diese können dann in einer neueren Version ins Programm eingefügt werden (Die Konfiguration steht in der Datei "~/.kde/share/config/kspaceduelrc" im Abschnitt "[Game]").

Die Parameter sind im einzelnen

  • Sonne:
    • Sonnenenergie: je höher, desto mehr Energie bekommen die Schiffe
    • Gravitation: Anziehungskraft der Sonne
  • Schuß:
    • Schußgeschwindogkeit: Abschußgeschwindigkeit der Kugeln
    • Energieverbrauch: Benötigte Energie für einen Schuß
    • maximale Anzahl: Maximale Anzahl Schüsse je Spieler auf dem Bildschirm
    • Schaden: Schaden beim Treffer
    • Lebensdauer: Maximale Lebensdauer eines Schusses
    • Ladezeit: Zeit die ein Schiff zum Nachladen eines Schusses benötigt
  • Mine:
    • Minentreibstoff: Energievorrat der Mine
    • Energieverbrauch: Benötigte Energie für das Legen einer Mine
    • Aktivierungszeit: je größer, desto länger dauert es, bis eine Mine aktiv wird
    • Schaden: Schaden beim Treffer
    • maximale Anzahl: Maximale Anzahl Minen je Spieler auf dem Bildschirm
    • Ladezeit: Zeit die ein Schiff zum Nachladen einer Mine benötigt
  • Allgemein:
    • Spielgeschwindigkeit: Regelt die Geschwindigkeit des gesamten Spiels
  • Start:
    • Position X und Position Y: Position der Schiffe zu Beginn einer Runde. Die Raumschiffe starten jeweils auf gegenüberliegenden Seiten der Sonne.
    • Geschwindigkeit X und Geschwindigkeit Y: Geschwindigkeit zu Beginn einer Runde
  • Schiff:
    • Beschleunigung: Beschleunigung der Schiffe
    • Energieverbrauch: Energieverbrauch beim Beschleunigen
    • Rotationsgschw.: Drehgeschwindigkeit der Schiffe
    • Energieverbrauch: Energieverbrauch beim Drehen
    • Schaden bei Zusammenstoß: Schaden beim Zusammenstoß beider Schiffe

3.3 Handikap

Jedem Spieler können mit "Optionen/Handikap..." die Trefferpunkte beim Start zugewiesen werden.

3.4 Graphik

Bildaufbauzeit: Dauer eines Bildaufbaus in ms. Alle Spielparameter sind unabhängig von der Bildaufbauzeit!
